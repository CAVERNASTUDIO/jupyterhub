# Reporting a Vulnerability / Reporte de Vulneración


If you believe you’ve found a security vulnerability in a Jupyter
project, please report it! /  ¿identifica una vulnerabilidad de seguridad en el proyecto Jupyter?, ¡Reporta!
See the / visita el portal [security documentation](https://jupyterhub.readthedocs.org/en/latest/contributing/security.html) for how.
